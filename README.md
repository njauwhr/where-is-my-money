
# Where Is My Money

一个用 Flutter 编写的个人记账应用示例，用于记录收入/支出、按分类管理账目，并用图表与日历视图做统计展示。

本仓库按你的要求只提供：

- `README.md`
- `app-release.apk`

## 功能概览

- 记一笔：新增收入/支出记录
- 分类管理：自定义分类，按分类查看与统计
- 统计图表：用图表查看支出结构与趋势
- 日历视图：按日期快速定位当天记录
- 本地存储：数据保存在本地 SQLite（离线可用）
